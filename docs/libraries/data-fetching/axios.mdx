# Axios

Axios is a popular JavaScript library for making HTTP requests from the browser or Node.js. It provides a simple and easy-to-use API for sending and receiving data over HTTP.

Axios allows you to create instances of the axios object with custom configuration options. This can be useful if you need to use different base URLs, headers, or interceptors for different parts of your application.

To create an instance of Axios, you can use the create method:

```ts
import axios, { AxiosInstance } from 'axios';

const instance: AxiosInstance = axios.create({
  baseURL: 'https://api.example.com',
  headers: {
    'Content-Type': 'application/json',
  },
});

// Send a GET request to /data
instance.get('/data')
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.error(error);
  });
```
